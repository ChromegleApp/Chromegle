Command List

  • Start a chat if one is not currently running with Omegle, start chat
  • Stop a chat if one is currently running with Omegle, stop chat

Privacy Info

This feature works by using Google's Web Speech API to convert your microphone input into text that Chromegle then parses into commands. Chromegle does not handle your voice data at all, meaning your voice messages are 100% confidential and not accessible by us.
